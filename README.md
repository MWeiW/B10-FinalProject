# HTW Campus Event and Workshop Hub - B10 Final Project

A small front-end web application for the course Web Application and Software Architecture at HTW Berlin.

## Project information

* Student: Wing Wei Wong
* Course: B10 Web Application / Software Architecture
* Scenario: Scenario 1 - Campus Event and Workshop Hub

## How to run

Open `index.html` directly in a web browser.

No installation or server is needed.

## Project idea

The website is a campus event and workshop hub. Guests can browse events, students can register for events, and organizer/admin users can manage their own events and view registrations.

## Demo roles

Use the role switcher at the top of the pages.

* Guest: browse and search events
* Student: register for events and manage own registrations
* Organizer/Admin: create, edit, and delete own events, and view registered students

Demo student users: Wei, Maya, Zoe
Demo organizer users: CS Department, Business School, Career Center

## Main features

* Event search and filters
* Event detail page
* Student registration form
* My Registrations page
* Organizer dashboard
* Event create, edit, and delete
* Registration create, view, edit, and cancel
* Registered-students table for organizers
* Role-based navigation
* Inline validation
* localStorage persistence

## Pages

* `index.html` - event list and filters
* `event.html` - event details and registration
* `registrations.html` - student registration management
* `event-form.html` - organizer event form
* `organizer.html` - organizer dashboard

## Technologies used

* HTML5
* CSS
* JavaScript
* jQuery
* JSON / JavaScript objects
* Browser localStorage

## Notes

This is a front-end only project. It uses sample data and browser localStorage instead of a real backend or database.

jQuery is used for form/filter handling and small visual effects.
